# Wedding Dress Code Checker

A practical, SEO-friendly wedding guest outfit tool built with Next.js.

## Features
- Main checker page at `/wedding-dress-code-checker`
- Rule-based recommendations by dress code, venue, season, time, and style
- FAQ + static SEO landing pages
- Metadata and basic schema markup

## Run
```bash
npm install
npm run dev
```

## Deploy
Recommended on Vercel.
